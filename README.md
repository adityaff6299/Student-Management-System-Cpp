
# Student Management System (C++)

A console-based Student Management System built using C++ with file handling for persistent data storage.

## Features
- Add, update, delete student records
- Search student by ID
- Display all records
- File handling using binary files
- Menu-driven interface

## Technologies Used
- C++
- File Handling (fstream)

## How to Run
1. Compile:
   g++ main.cpp -o sms
2. Run:
   ./sms
